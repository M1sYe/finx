# Finx - Music Downloader

A lightweight, CLI-based utility for downloading music. Built with simplicity and efficiency in mind.

## Features

- Search tracks
- Download audio in MP3 format (192kbps quality)
- Simple and intuitive CLI interface
- Fast downloads with Yt-dlp backend
- Add album covers

## Requirements

- **Python 3.10 or higher**
- **FFmpeg** (for Alpha v0.3 and lower)

## Build
- Open programm directory
```
python -m venv venv
\venv\Scripts\activate #For powershell: Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass
```
- Download **dependencies**
```
in shell:
python -m pip install -r requirements.txt
```
- **Build** file build.bat
